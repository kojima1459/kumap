# Manus 1兆トークンキャンペーン応募文

## 渾身の1パターン：評価基準完全対応型

【創造性×実用性×実行×Manus熟練】全部入りの社会課題解決アプリ、完成しました。

「クマップ」は、全国に散在するクマ出没情報を統合した日本初の特化型マップです。

**なぜ作ったか？**
クマ被害が過去最多を更新する中、情報は都道府県ごとにバラバラ。Yahoo!ニュース、20種類の公式マップ（Google Maps、ArcGIS、PowerBI等）を横断しないと全体像が見えない。この「情報の分断」が命に関わる問題だと気づきました。

**何を実現したか？**
✓ 毎日午前2時、Yahoo!ニュースから自動スクレイピング
✓ 20都道府県の公式マップURLを統合、ワンクリックアクセス
✓ 公式データ（赤マーカー）×ユーザー投稿（橙マーカー）の二層構造
✓ 登録地域への新情報プッシュ通知
✓ 期間・地域フィルター、統計ページ完備

**Manusの力を最大活用**
・Google Maps統合で位置情報を直感的に可視化
・node-cronで定期スクレイピング自動化
・tRPC+Drizzleでリアルタイム同期
・OAuth認証+通知システムでユーザー体験を最適化

技術と社会性を両立させた、命を守るインフラです。

## 評価基準への対応詳細

### 創造性 ⭐⭐⭐⭐⭐
- **独自性**: 日本初のクマ出没情報特化型統合マップ
- **アイデアの新規性**: 散在する20種類の異なる技術（Google Maps、ArcGIS、PowerBI等）の情報を単一UIに集約
- **差別化要素**: 公式データ×ユーザー投稿の二層構造で情報の鮮度と網羅性を両立

### 実用性 ⭐⭐⭐⭐⭐
- **現実世界の影響**: クマ被害が過去最多（2024年度）を記録する中、命を守る直接的な社会貢献
- **ターゲットユーザー**: 登山者、農業従事者、山間部住民、自治体職員、報道関係者
- **即座の価値提供**: 登山前・散歩前の安全確認、子どもの通学路チェック、自治体の被害予測に即活用可能

### 実行 ⭐⭐⭐⭐⭐
- **デザイン品質**: Google Maps統合による直感的UI、色分けマーカー（公式=赤/ユーザー=橙）で情報源を明確化
- **UX設計**: 期間フィルター（1週間/1ヶ月/3ヶ月）、都道府県別フィルター、統計ランキングで多角的分析
- **技術的完成度**: 毎日自動スクレイピング、重複排除ロジック、通知システム、レスポンシブ対応

### Manusの熟練 ⭐⭐⭐⭐⭐
- **フル機能活用**:
  - Google Maps API統合（マーカー、情報ウィンドウ、ジオコーディング）
  - データベース（Drizzle ORM）で141件の出没記録を管理
  - tRPCでフロントエンド↔バックエンドの型安全な通信
  - OAuth認証でユーザー管理
  - 通知API（Manus Forge）で登録地域への自動アラート
  - node-cronでスケジュール実行（毎日午前2時JST）
  - Cheerioで動的スクレイピング
- **技術スタック**: React 19 + TypeScript + Tailwind 4 + Express + MySQL + Node.js
- **運用設計**: エラーハンドリング、重複チェック、ログ記録、管理者ページ完備
